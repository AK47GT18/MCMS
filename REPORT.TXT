You are a senior full-stack engineer. Build a complete, production-ready Report Builder module for the RCMS (Road Construction Management System) using Node.js, Prisma, PostgreSQL, vanilla HTML/CSS/JS (no frontend frameworks).

Backend: Node.js + Express

json{

json{

}
Implement every report below as a Prisma query with dynamic where, select, and optional groupBy. Each report maps to real Prisma models from the schema.
GET /api/reports/catalog
Returns the full report catalog filtered by the calling user's role (read role from req.user.role).
